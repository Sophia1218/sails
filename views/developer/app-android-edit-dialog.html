Edit allowed Android applications

This API key will be deployed into Sails Android SDK using in your Android application.
API requests are sent directly to Sails from your client Android device. Sails verifies that each request originates from an Android application that matches one of the certificate SHA1 fingerprints and package names listed below. You can discover the SHA1 fingerprint with base64 encode of your developer certificate by keytool using the following command:
keytool -exportcert -alias -keystore | openssl sha1 -binary | openssl base64
One SHA1 certificate fingerprint with base64 encode and package name (separated by a semicolon) per line. Example:
478uEnKQV+fMQT8Dy4AKvHkYibo=;com.example
